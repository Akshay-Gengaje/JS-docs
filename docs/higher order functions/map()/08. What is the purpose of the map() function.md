# What is the purpose of the map() function?

The `map()` function in JavaScript is a **higher-order function** used to **transform** each element in an array **without modifying** the original array. It **returns a new array** where each element is the result of applying a callback function to the corresponding element in the original array.

## **ğŸ“Œ Key Characteristics of `map()`**
1. âœ… **Creates a new array** (does not modify the original array).
2. âœ… **Iterates over each element** and applies a callback function.
3. âœ… **Requires a return value** (otherwise, `undefined` is stored in the new array).
4. âœ… **Keeps the same array length** (unlike `filter()` which may remove elements).
5. âœ… **Does not mutate the original array** (pure function concept in functional programming).

   - `currentValue` (required) â†’ The current element being processed.
   - `index` (optional) â†’ The index of the current element.
   - `array` (optional) â†’ The original array.
2. `thisArg` (optional) â†’ Value to use as `this` inside the callback.

### **Returns:**  
ğŸ”¹ A **new array** with transformed elements.
