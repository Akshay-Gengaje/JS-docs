# How does reduce() work in JavaScript?

The `reduce()` method in JavaScript **iterates over an array and accumulates values into a single result**. Unlike `map()` (which returns an array) or `filter()` (which returns a subset of elements), `reduce()` **returns a single value**, which could be a number, object, array, or even a string.

---

## **âœ… Syntax of `reduce()`**
```javascript

    // Process each element and update the accumulator
}, initialValue);
```

### **ðŸ”¹ Parameters**
1. **`accumulator`** â†’ Stores the accumulated result from previous iterations.  
2. **`currentValue`** â†’ The current element being processed.  
3. **`currentIndex`** (optional) â†’ The index of `currentValue`.  
4. **`array`** (optional) â†’ The original array that `reduce()` is called on.  
5. **`initialValue`** (optional) â†’ The starting value of `accumulator`. If omitted, the first array element is used as `accumulator`, and iteration starts from index `1` instead of `0`.
