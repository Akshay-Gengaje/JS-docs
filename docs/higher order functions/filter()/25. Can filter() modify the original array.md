# Can filter() modify the original array?

No, the **`filter()` method does not modify the original array**. It creates and returns a **new array** containing only the elements that pass the provided test function.

1. **Returns a new array** â†’ The original array remains unchanged.  
2. **Does not mutate elements** â†’ It only selects elements that meet the condition.  
3. **Shallow copy** â†’ It creates a new array but does not deeply copy objects inside it.
