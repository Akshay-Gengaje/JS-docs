# What is function composition?

ğŸ“Œ **Function composition** is a technique in functional programming where multiple **small functions are combined** to create a new function.  
ğŸ“Œ The output of one function becomes the **input of the next function**, allowing **cleaner, reusable, and modular** code.  
ğŸ“Œ It follows the **"compose small functions to build complex logic"** approach instead of writing long functions.  

---

## **ğŸ”¹ Example of Function Composition**
### **âœ… Without Composition (Traditional Approach)**
```javascript
function capitalize(str) {
    return str.toUpperCase();
}

function addExclamation(str) {
    return str + "!";
}

function greet(name) {
    return "Hello, " + name;
}

const result = addExclamation(capitalize(greet("Alice")));
console.log(result); // Output: HELLO, ALICE!
```
ğŸ“Œ Here, we manually **nest** function calls, making it harder to read.  

---

### **âœ… Using Function Composition**
```javascript
function compose(f, g) {
    return function (x) {
        return f(g(x)); // Apply g() first, then f()
    };
}

const greetAndShout = compose(addExclamation, compose(capitalize, greet));

console.log(greetAndShout("Alice")); // Output: HELLO, ALICE!
```
ğŸ“Œ Now, the function execution **flows naturally** from right to left.

## **ğŸ”¹ Function Composition with Multiple Functions**
Instead of just two functions, we can **compose multiple functions dynamically**.  

### **âœ… Generalized Function Composition**
```javascript
const compose = (...functions) => (input) => 
    functions.reduceRight((acc, fn) => fn(acc), input); // Apply functions from right to left

const finalGreet = compose(addExclamation, capitalize, greet);

console.log(finalGreet("Alice")); // Output: HELLO, ALICE!
```
âœ… **How It Works?**  
- `reduceRight()` applies functions **from right to left**, like a pipeline.  
- Functions execute in this order:  
  **`greet("Alice") â†’ capitalize() â†’ addExclamation()`**  

---

## **ğŸ”¹ Pipe: Left-to-Right Composition**
ğŸ”¹ **`compose()`** runs functions from **right to left** (`f(g(x))`).  
ğŸ”¹ **`pipe()`** runs functions from **left to right** (`g(f(x))`).  

```javascript
const pipe = (...functions) => (input) => 
    functions.reduce((acc, fn) => fn(acc), input);

const finalGreet = pipe(greet, capitalize, addExclamation);

console.log(finalGreet("Alice")); // Output: HELLO, ALICE!
```
âœ… **Key Difference**:  
- **`compose()` â†’ right to left (`f(g(h(x)))`)**  
- **`pipe()` â†’ left to right (`h(g(f(x)))`)**
